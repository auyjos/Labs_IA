# Labs de Inteligencia Artificial

Este repositorio contiene los laboratorios del curso de Inteligencia Artificial.

## Estructura del Repositorio

Cada laboratorio está en su propia branch:

- `lab4` - Laboratorio 4: Implementación "Bare Metal" de algoritmos de ML
  - Gradient Descent (Batch, SGD, Mini-batch)
  - Perceptrón para clasificación binaria

## Cómo usar este repositorio

Para acceder a un laboratorio específico, cambia a la branch correspondiente:

```bash
git checkout lab4
```

## Tecnologías

- Python 3.x
- NumPy
- Pandas
- Matplotlib
- Jupyter Notebooks
